# C1901 产品介绍

![C1901 Banner](/img/c1901-banner.png)

## 产品简介

Jetson Orin Nano Lite **(C1901)** 是一款针对 **Jetson Orin Nano & NX** 核心卡进行开发的高性价比载板，该底板基于公版方案进行开发，对公版的部分功能进行删减和优化，在保证不影响基础功能的情况下，大幅度提升性价比，助力客户降本增效。

## 产品特点

- C1901基于原厂方案进行开发，提供接近公版的体验
- 支持 Orin Nano 4G/8G & Orin NX 8G/16G
- 高品质6层沉金PCB，全MLCC电容以及镀金连接器
- 支持 NVME 硬盘和 M.2 网卡

## 版本迭代记录

| 版本 | 日期 | 备注 |
|------|------|------|
| V1.0 | 2023 Q3 | C1901的第一个版本 |
| V1.1 | 2024 Q1 | 调整了功耗墙，该问题曾导致载板在使用 16G Orin NX 核心卡在极限负载情况下，断电重启的问题 |
| V1.2 | 2024 Q2 | 增加 RTC 电池座子 |
| V1.3 | 2025 Q1 | 1. 修改了 CSI 相机接口与原厂载板一致 2. 修复了一个 GPIO 问题（12/35/38/40 号引脚） 3. 修复了风扇相关问题 4. 移除了所有 I2C 接口的上拉电阻 5. 不再提供对 Jetson Nano/Xavier NX/TX2 NX 等停产核心卡的支持 6. 移除载板硬件功耗墙 |

> V1.3 版本需要使用专用的设备树文件。V1.2 版本之前的设备树不影响使用，V1.2 版本推荐使用 5.1.4 版本的固件。

## 产品规格

| 规格 | 参数 |
|------|------|
| 供电电压 | 12V（兼容9-19V） |
| 工作温度 | 0℃~45℃ |
| 支持的核心 | Orin Nano 4G/8G, Orin NX 8G/16G |
| 电源接口 | DC5521（兼容DC5525插头） |
| 以太网 | 1x 千兆网口（RJ45） |
| 无线网卡 | 1x M.2 Key E |
| 存储扩展 | 2x M.2 Key M (NVME SSD) |
| 显示 | 1x HDMI (1080P) |
| USB | 3x USB3.0 + 1x USB2.0 |
| USB C | 1x Type C (USB2.0 仅烧录用) |
| 相机接口 | 2x CSI-22P |
| 其余接口 | 和原厂一致 |
| RTC电池 | 支持 |
| POE供电 | 不支持 |

> - 本底板支持 12V(9V~19V) 供电，请保证电源拥有足够的功率
> - 电源接口为 DC5521，兼容 DC5525 插头
> - 当供电电压大于 15V 且负载较大时，请注意加强散热
> - 本产品支持电源输入反接保护
> - 每组 USB 接口合计最大输出电流为 2.5A
> - 背面网卡接口与 M.2 2230 接口相比原厂改为对称布局

## 部分功能说明

- 载板在某些特定功率负载下可能会产生啸叫声，该现象由电感导致，不影响系统正常工作
- HDMI 接口由 DP 接口转接而来，在原厂固件下仅支持 1080P 分辨率
- 40Pin GPIO 排针中的 5V 输出具有倒灌保护，最大电流 0.7A
- 固态硬盘必须使用单面版本，双面版本可能发生干涉引起短路
- RTC 电池连接器型号 MX1.25-2P，电池电压 3V，禁止接反
