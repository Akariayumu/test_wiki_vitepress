# C1902 产品介绍

![C1902 Banner](/img/c1902-banner.png)

## 产品简介

**C1902** 是一款针对 **Jetson Orin Nano & NX** 核心卡进行研发的载板，该底板基于公版方案进行开发，完美兼容官方设计套件的功能，包括全速的 USB3.0 接口及 TYPE-C 接口。

:::tip 核心特性
- 基于原厂方案开发，可直接烧录 **Developer Kit 固件**
- 支持 Orin Nano 4G/8G & Orin NX 8G/16G
- 高品质 6 层沉金 PCB，全 MLCC 电容以及镀金连接器
- 支持 NVME 硬盘和 M.2 网卡
:::

## 产品特点

1. **C1902** 基于原厂方案进行开发，所有接口功能与原厂一致且可以 **直接烧录 Developer Kit 固件**。
2. 支持 **Orin Nano 4G/8G & Orin NX 8G/16G**。
3. 高品质 6 层沉金 PCB，全 MLCC 电容以及镀金连接器。
4. 支持 **NVME 硬盘** 和 **M.2 网卡**。

### 注意事项

:::warning 重要
- 本底板支持 12V (9V~19V) 供电，请保证电源功率充足。
- 电压大于 15V 且负载较大时，请加强散热。
- 支持电源反接保护，但任何时候都不应反接电源。
- USB 每组最大 2.5A，过流自动切断。
- 背面网卡与 M.2 2230 布局与原厂对称相反。
:::

:::danger 禁止
禁止在载板未断电时拔插核心卡，会导致损坏，后果自负。
:::

## 部分功能说明

### 启动按钮

- **系统启动时**：短按进入关机界面，长按 10s 强制关机。
- **系统关闭时**：短按开机。

### 40PIN 排针

40Pin GPIO 的 5V 输出有倒灌保护，最大 0.8A，过流关断。

![40PIN排针](/img/c1902-40pin.png)

### 12Pin 调试接口

![12Pin调试接口](/img/c1902-12pin.png)

### RTC 电池

MX1.25-2P 连接器，电池电压 3V。**注意极性，禁止反接。**

![RTC电池](/img/c1902-rtc.png)

:::tip
接入电池后需配置才能启用 RTC 时钟。
:::

:::warning SSD 注意
固态硬盘必须使用**单面版本**，双面可能干涉短路。
:::
