# C2401 产品介绍

![C2401 Banner](/img/c2401-banner.png)

## 产品简介

**C2401** 是一款针对 **Jetson Orin Nano** 及 **NX** 核心卡进行研发的高密度小尺寸载板。在保证小体积的同时拥有完备的接口，如**双网口**、**双 USB 接口**，以及丰富的拓展能力，能拓展**4G/5G 模块**等接口。

## 产品特点

- 配备先进的车规级 E-fuse 芯片，过压、欠压、短路、过流、反接保护
- 选用超低内阻 Buck 芯片，合金扁线电感，低发热、高饱和电流
- 8层 2u 沉金塞孔工艺 PCB，外围尺寸仅 **55x92mm**
- 双 USB3.0，双网口（含 2.5G 支持 PTP + 16KB 巨型帧）
- M.2 接口同面设计，更换网卡/硬盘无需拆除核心卡

## 产品规格

| 规格 | 参数 |
|------|------|
| 供电电压 | 12V 兼容 (9.5V-30V) |
| 电源接口 | XT30 接口 |
| 支持的核心 | Orin Nano 4G/8G, Orin NX 8G/16G |
| 以太网 | 1x 千兆（9KB 巨型帧）+ 1x 2.5G（16KB 巨型帧、PTP） |
| 无线网卡 | 1x M.2 Key E（2230） |
| 存储扩展 | 1x M.2 Key M (2230 NVME SSD) |
| 显示 | 1x Mini HDMI V2.1 |
| USB | 2x USB3.2 (10Gbps), 1x USB2.0 (GH1.25-4p) |
| USB C | 1x Type C (USB2.0) 烧录固件用 |
| 相机接口 | 1x CSI-22P |
| 其余接口 | UART、CAN、I2C |
| RTC 电池 | 1x RTC 后备时钟电池接口 |
| 工作温度 | 0℃~45℃ |

> - 本底板支持 12V (9.5V~30V) 供电，电压大于 15V 时请注意散热
> - 支持电源输入反接保护
> - 每组 USB 接口最大输出电流 2.5A
> - **禁止在载板未断电时拔插核心卡**

## 部分功能及接口介绍

### 电源接口 (XT30)

![C2401 XT30 电源](/img/c2401-xt30.png)

推荐输入电压 12V，支持 10V-30V。欠压保护 9.5V，过压保护 31V。

| 核心卡 | 推荐电源功率 | USBC-PD 需支持 |
|--------|-------------|---------------|
| ORIN NANO 4G/8G | 45W | 20V 2.25A |
| ORIN NX 8G/16G | 60W | 20V 3A |

### REC 按钮与启动

- **进入恢复模式：** 断电时按住 REC 按钮再通电；或将拨码开关 3 拨至 ON（每次上电自动进入）
- **启动按钮：** 开机时按一次关机 / 长按 10 秒强制关机；关机时按一次开机
- 拨码开关 2 拨至 ON 可禁用上电自动开机
