# C1902 Introduction

![C1902 Banner](/img/c1902-banner.png)

## Introduction

The **C1902** is a carrier board specifically designed for **Jetson Orin Nano & NX** modules. Built on a standard board design, it fully supports the official developer kit's features, including full-speed USB3.0 and Type-C ports.

:::tip Key Features
- Developed based on the original manufacturer's solution — supports **direct Developer Kit firmware flashing**
- Compatible with Orin Nano 4G/8G & Orin NX 8G/16G
- Premium 6-layer gold-plated PCB with full MLCC capacitors and gold-plated connectors
- Supports M.2 NVMe SSD and M.2 Wi-Fi module
:::

## Features

1. All interface functions match the original manufacturer's and support direct Developer Kit firmware flashing.
2. Compatible with NVIDIA® Jetson **Orin Nano (4GB/8GB)** and **Orin NX (8GB/16GB)** modules.

### Important Notes

:::warning Important
- Requires 12V (9-19V) DC supply with sufficient current.
- Provide adequate cooling when voltage > 15V under high load.
- Reverse-polarity protection included — but never reverse-connect.
- USB OCP threshold is 2.5A per group.
- Back-side M.2 layout is symmetrical (differs from original).
:::

:::danger Warning
Do not hot-swap the module. Always power off first. Damage from improper operation is not covered under warranty.
:::

## Feature Details

### Power Button

- **System Running**: Short press → shutdown menu, long press 10s → force power-off.
- **System Off**: Short press → power on.

### 40-Pin GPIO Header

5V output with reverse current protection, max 0.8A, OCP auto-disables on fault.

![40-Pin Header](/img/c1902-40pin.png)

### 12-Pin Debug Header

![12-Pin Debug](/img/c1902-12pin.png)

### RTC Battery

MX1.25-2P connector, 3V battery. **Mind polarity — do not reverse.**

![RTC Battery](/img/c1902-rtc.png)

:::tip
Configuration required after connecting battery to enable RTC clock.
:::

:::warning SSD Compatibility
Only **single-sided** SSDs are supported. Double-sided may cause interference and short circuits.
:::
