# 安装使用ollama

##### 1. 运行以下指令安装ollama

```shell
curl -fsSL https://ollama.com/install.sh | sh
```

##### 2. 检查是否正确安装

```shell
jetson@jetson-desktop:~$ ollama -v
ollama version is 0.9.6
```

##### 3. 下载并运行deepseek-r1 1.5b模型

```
ollama run deepseek-r1:1.5b
```
